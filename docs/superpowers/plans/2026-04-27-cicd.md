# CI/CD Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Auto-deploy BE-Moon lên VPS mỗi khi push lên `main`, với manual trigger tùy chọn.

**Architecture:** GitHub Actions SSH vào VPS, chạy `git pull → yarn install → pm2 restart galaxy`. Secrets đã được cấu hình sẵn trên GitHub repo.

**Tech Stack:** GitHub Actions, `appleboy/ssh-action@v1.2.0`, PM2, Yarn

---

### Task 1: Tạo GitHub Actions workflow

**Files:**
- Create: `.github/workflows/deploy.yml`

- [ ] **Step 1: Tạo thư mục `.github/workflows/`**

```bash
mkdir -p .github/workflows
```

- [ ] **Step 2: Tạo file `deploy.yml`**

Tạo file `.github/workflows/deploy.yml` với nội dung sau:

```yaml
name: Deploy to VPS

on:
  push:
    branches: [main]
  workflow_dispatch:

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - name: Deploy via SSH
        uses: appleboy/ssh-action@v1.2.0
        with:
          host: ${{ secrets.VPS_HOST }}
          username: ${{ secrets.VPS_USER }}
          key: ${{ secrets.VPS_SSH_KEY }}
          port: ${{ secrets.VPS_PORT }}
          script: |
            cd ${{ secrets.VPS_APP_PATH }}
            git pull origin main
            yarn install --frozen-lockfile
            pm2 restart galaxy
```

- [ ] **Step 3: Commit và push**

```bash
git add .github/workflows/deploy.yml
git commit -m "ci: add GitHub Actions deploy workflow"
git push
```

- [ ] **Step 4: Xác nhận workflow chạy thành công**

1. Vào repo GitHub → tab **Actions**
2. Chờ workflow `Deploy to VPS` hoàn thành (màu xanh ✓)
3. Nếu thất bại, click vào job để xem log lỗi

**Lỗi thường gặp và cách fix:**

| Lỗi | Nguyên nhân | Fix |
|-----|-------------|-----|
| `Permission denied (publickey)` | Public key chưa được add lên VPS | Chạy `ssh-copy-id -i ~/.ssh/vps_deploy.pub user@VPS_IP` |
| `Host key verification failed` | Lần đầu SSH vào host | Thêm `script: ssh-keyscan $HOST >> ~/.ssh/known_hosts` trước bước deploy |
| `pm2: command not found` | PATH không có pm2 khi SSH non-interactive | Đổi thành `/usr/local/bin/pm2 restart galaxy` hoặc `~/.yarn/bin/pm2 restart galaxy` |
| `yarn: command not found` | Tương tự PATH issue | Dùng full path: `/usr/local/bin/yarn install --frozen-lockfile` |
| `cd: no such file or directory` | `VPS_APP_PATH` sai | Kiểm tra lại secret `VPS_APP_PATH` |

- [ ] **Step 5: Test manual trigger**

1. Vào GitHub → tab **Actions** → workflow `Deploy to VPS`
2. Click **Run workflow** → chọn branch `main` → **Run workflow**
3. Xác nhận workflow chạy thành công
